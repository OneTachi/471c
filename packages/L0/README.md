L0 adds the ability to use functions in the language by listing a Program as a series of Procedures. Each Procedure boils down into a Python function. The language retains the same overall characteristics as L1 with the statements variable; however, it has an additional class labeled Address. This is a higher level function is essentially equivalent to C's `int i = 10`. It does Copy, but is given a value directly rather than copying from another variable.
